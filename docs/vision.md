# Vision

## One-line vision

A desktop tool that turns Japanese gameplay text into clean, reviewable Anki cards.

## Product philosophy

This project should not try to replace the learner's judgment.

The system should help with:

- capture
- extraction
- tokenization
- enrichment
- card formatting

The learner should stay in control of:

- what to learn
- whether the OCR result is correct
- which token matters
- which meaning to keep
- whether the card should be added at all

## Core principle

**Automation supports the decision. Automation does not make the decision.**

## Who this is for

Primary audience:

- learners of Japanese
- especially people learning through games
- especially people playing text-heavy games such as Pokémon

Secondary audience:

- learners who already use Anki
- people who want contextual vocabulary mining without a complicated setup

## Product goals

1. Make it fast to turn a screenshot into an Anki card.
2. Keep the review step central.
3. Stay friendly to install and run.
4. Keep the app usable as a local desktop tool.
5. Structure the project so contributors can understand it quickly.

## Non-goals for the first version

- perfect OCR in every game
- full automation
- cloud sync
- mobile support
- batch mining from videos
- advanced linguistics features
- highly customizable card templates from day one

## Success criteria

The project is successful when a user can go from screenshot to reviewed Anki card quickly and with confidence.

Ideal loop:

- load screenshot
- review OCR text
- click a token
- adjust card fields
- add to Anki

The experience should feel simple, not technical.
